Phonebook

Phonebook is my personal project. It is a full-stack application with CRUD functionality for managing phone contacts.

The frontend is created with React.js, the backend is implemented on top of Node.js with the help of the Express library, and the application's data is stored in a MongoDB database.

The app provides users with the following functionality:

  • Adding new contacts to the phonebook (each entry consists of a first name, last name, and phone number). If a number is added to an already existing contact, the new number replaces the old number;
  • Deleting existing contacts from the phonebook;
  • Filtering the list of contacts by keyword;
  • Filtering the list of contacts by the first letter of the first name (by clicking on the corresponding alphabetical tabs).

Designed and coded with by Kseniia Priakhina. Built with HTML, CSS, and Bootstrap.
